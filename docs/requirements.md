# System Requirements

## Minimum Baseline
- **CPU:** Quad-core x86_64 processor (Intel i5/Ryzen 5 class) or Apple Silicon M1
- **GPU:** Optional; integrated graphics acceptable
- **RAM:** 16 GB
- **Storage:** 256 GB SSD (NVMe recommended)

## Recommended Baseline
- **CPU:** 8-core x86_64 processor (Intel i7/Ryzen 7 class) or Apple Silicon M2 Pro
- **GPU:** NVIDIA RTX 3060 (12 GB VRAM) or Apple Silicon GPU equivalent
- **RAM:** 32 GB
- **Storage:** 1 TB NVMe SSD

## Optional Accelerators
- **High-end GPU:** NVIDIA RTX 4090, A6000, or L40S; AMD MI210; or Apple M3 Max GPU
- **Inference accelerators:** AWS Inferentia, Google TPU v3+, Habana Gaudi2
- **Vector search:** Dedicated PGVector-ready PostgreSQL instance or managed vector database (Pinecone, Weaviate)

## Networking
- **Minimum bandwidth:** 50 Mbps symmetric
- **Recommended bandwidth:** 200 Mbps symmetric with low latency VPN access for remote collaboration

## Operating Systems
- Ubuntu 22.04 LTS or later, macOS 13 Ventura or later, Windows 11 with WSL2 (Ubuntu 22.04)
